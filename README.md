# 🐍 Python Mini Projects

A collection of beginner-friendly Python projects created while learning Python programming. These projects helped me improve my understanding of Python fundamentals such as loops, conditions, functions, modules, and user input.

## 📂 Projects Included

### ✅ To-Do List
A simple command-line To-Do List application that allows users to:
- Add tasks
- View tasks
- Remove tasks
- Manage daily activities

**File:** `To-Do-List.py`

---

### 🎯 Number Guessing Game
A fun game where the computer randomly selects a number and the player tries to guess it.

**Features**
- Random number generation
- User input validation
- Hint messages (Too High / Too Low)

**File:** `number-guessing-game.py`

---

### 🔐 Password Generator
Generate strong and secure random passwords.

**Features**
- Custom password length
- Random letters, numbers, and symbols
- Easy to use

**File:** `password_generate.py`

---

### ✂️ Rock Paper Scissors
A classic Rock-Paper-Scissors game where you play against the computer.

**Features**
- Computer-generated moves
- Winner detection
- Interactive gameplay

**File:** `rock-paper-scissors.py`

---

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed

### Clone the Repository

```bash
git clone https://github.com/manmohan-01/Python-Mini-Projects.git
```

### Navigate to the Project Folder

```bash
cd Python-Mini-Projects
```

### Run Any Project

Example:

```bash
python To-Do-List.py
```

or

```bash
python number-guessing-game.py
```
